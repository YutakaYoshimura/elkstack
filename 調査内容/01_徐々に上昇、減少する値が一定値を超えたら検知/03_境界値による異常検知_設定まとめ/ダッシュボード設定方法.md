# Kibana 8.17.0 Lens で境界線を表示する手順

## 前提条件

| 項目 | 値 |
|---|---|
| Kibana バージョン | 8.17.0（日本語UI） |
| データソース | `.ml-anomalies-shared` |
| 対象ジョブ | `vibration-anomaly-detection` |
| 表示するフィールド | `actual`・`upper_confidence_bound`・`lower_confidence_bound` |

---

## 手順

### 1. Lens を開く

```
ダッシュボード
→「作成」
→「パネルを追加」
→「レンズ」を選択
```

---

### 2. データビューを変更する

```
左上のドロップダウン
→「.ml-anomalies-shared」を選択
```

---

### 3. フィルターを追加する

画面上部「**フィルターを追加**」を2回クリックして以下を設定します。

**1つ目**

| 項目 | 値 |
|---|---|
| フィールド | `job_id` |
| 演算子 | 次と等しい |
| 値 | `vibration-anomaly-detection` |

**2つ目**

| 項目 | 値 |
|---|---|
| フィールド | `result_type` |
| 演算子 | 次と等しい |
| 値 | `record` |

---

### 4. グラフタイプを選択する

```
右側パネル上部のグラフ種類一覧
→「折れ線グラフ」を選択
```

---

### 5. 横軸を設定する

```
右側パネル「横軸」をクリック
→「timestamp」を選択
→「最小間隔」に「900s」と入力
→「閉じる」
```

---

### 6. 縦軸に3つのフィールドを追加する

右側パネル「**縦軸**」の「**フィールドを追加またはドラッグアンドドロップ**」を3回クリックして順番に設定します。

**① 実測値（actual）**

```
左側フィールド一覧で「actual」を検索して選択
→ 集計方法 :「最大値」
→ 表示名   :「実測値」
→「閉じる」
```

**② 上の境界**

```
「anomaly_score_explanation.upper_confidence_bound」を検索して選択
→ 集計方法 :「最大値」
→ 表示名   :「上の境界」
→「閉じる」
```

**③ 下の境界**

```
「anomaly_score_explanation.lower_confidence_bound」を検索して選択
→ 集計方法 :「最小値」
→ 表示名   :「下の境界」
→「閉じる」
```

---

### 7. 色を設定する

縦軸に追加した各フィールドの右側にある**鉛筆アイコン**をクリックして色を変更します。

| 項目 | 色 |
|---|---|
| 実測値 | 青 |
| 上の境界 | 赤 |
| 下の境界 | オレンジ |

※ Lens では線種（破線・実線）の変更ができないため、色で区別します。破線を使いたい場合は TSVB を使用します。

---

### 8. 時間範囲を調整する

データが表示されない場合は右上の時間範囲を変更します。

```
右上の時間範囲ピッカー
→「過去30日間」に変更
```

---

### 9. 保存する

```
右上「保存して戻る」をクリック
→ ダッシュボードに戻る
→「ダッシュボードを保存」
```

---

## フィールドが候補に出ない場合

`anomaly_score_explanation.upper_confidence_bound` が左側フィールド一覧に表示されない場合は以下を実施します。

```
Stack Management（スタック管理）
→ Kibana
→「データビュー」
→「.ml-anomalies-shared」を選択
→ 右上「フィールドリストを更新」をクリック
→ Lens に戻って再試行
```

---

## 補足：破線で境界線を表現したい場合

Lens では線種の変更ができません。破線で境界線を表現したい場合は **TSVB** を使用します。

| ツール | 線種変更 | 設定の手軽さ |
|---|---|---|
| Lens | 不可（色のみ） | 簡単 |
| TSVB | 可能（破線・実線） | やや複雑 |
